Replication Package:  Health Effects of Bridge Employment
Author: Sungsik Hwang, Shiro Furuya, and Jenna Nobles
Contact: shwang97@wisc.edu
Date: 2025-08-04

The replication package contains three separate do-files:
BE_cleaning – prepares the data for the analysis.
BE_MainAnalysis – contains the main analysis of clinical/subclinical health outcomes and health-related behaviors
BE_Robustness – includes robustness checks in the Supplementary Figures/Tables.
